System Architecture

Hardware: KrakenSDR 5-Channel SDR / RSPduo Dual-Tuner
         ├─ Ch0: Reference (illuminator-facing antenna)
         └─ Ch1-4: Surveillance (target-facing array)
                   │
         Phase Calibration (noise source GPIO control)
                   │
         Conditioning (AGC with decay-only mode)
                   │
         Block B3 Reconstructor (FM/ATSC3/DVB-T) ← 10-20 dB improvement
                   │
         ECA-B Clutter Canceller (NLMS adaptive)
                   │
         CAF Processing (2D FFT cross-ambiguity)
                   │
         Doppler Processing (slow-time FFT)
                   │
         CFAR Detection (CA/GO/SO/OS, O(n) prefix-sum)
                   │
         Detection Clustering (8-connected components)
                   │
         AoA Estimation (Bartlett/MUSIC)
                   │
         Multi-Target Tracker (Kalman + GNN)
                   │
         Display System (Tkinter + matplotlib)

Signal Processing Blocks

krakensdr_source

5-channel coherent SDR source with direct libusb GPIO noise source control for calibration.

Details →

rspduo_source

SDRplay RSPduo in dual-tuner mode. High dynamic range, IF gain 20-59 dB.

Details →

conditioning

AGC with decay-only mode, clamped gain (max 1000), handles silence gracefully.

CPU GPU Details →

dvbt_reconstructor Block B3

Reference signal reconstruction: FM demod-remod, ATSC3 OFDM with SVD pilot enhancement.

Details →

eca_canceller

NLMS adaptive clutter cancellation. Removes direct path and multipath interference.

CPU GPU 10x Details →

caf_processing

Cross-Ambiguity Function via 2D FFT. Generates range profiles.

CPU GPU 23x Details →

doppler_processor

Slow-time FFT for Doppler extraction. Windowing: Hamming/Hann/Blackman.

CPU GPU 1.2x Details →

cfar_detector

2D CA/GO/SO/OS-CFAR with O(n) prefix-sum optimization. Pfa-calibrated threshold.

CPU GPU 305x Details →

detection_cluster

8-connected component clustering for grouping adjacent detections.

Details →

aoa_estimator

Angle-of-Arrival estimation: Bartlett beamforming or MUSIC (Eigen3 SVD).

Details →

tracker

Multi-target Kalman tracker with Global Nearest Neighbor (GNN) association.

Details →

coherence_monitor

Phase coherence tracking across channels. Triggers calibration when drift detected.

Build & Install

# Install dependencies
sudo apt install cmake libfftw3-dev libeigen3-dev gnuradio-dev python3-pybind11

# Build with GPU support
mkdir build && cd build
cmake .. -DCMAKE_BUILD_TYPE=Release -DENABLE_GPU=ON
make -j$(nproc)
sudo make install && sudo ldconfig

# Run tests
pytest tests/ -v
